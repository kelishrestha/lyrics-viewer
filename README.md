# Lyrics viewer

Simple web-based application in Vite React to view lyrics in floating mode.

## Dependencies

- [Vite v7.0.0+](https://vitejs.dev/)
- [React v19.2.0+](https://reactjs.org/)
- [Typescript v5.9.3](https://www.typescriptlang.org/)

## Install

```
npm install
```

## Build

For development, run
```
npm run dev
```

For production, run
```
npm run build
```

For preview mode, run
```
npm run preview
```
